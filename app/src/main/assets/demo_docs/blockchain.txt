Блокчейн и криптовалюты

Блокчейн (цепочка блоков) - это распределённая база данных, которая хранит упорядоченные записи (блоки) в хронологическом порядке. Каждый блок связан с предыдущим криптографически.

Принцип работы блокчейна
Блокчейн состоит из цепочки блоков, где каждый блок содержит:
1. Данные транзакций
2. Временную метку
3. Хеш предыдущего блока
4. Собственный хеш

Криптографический хеш
Хеш-функция преобразует данные любого размера в строку фиксированной длины. SHA-256 используется в Bitcoin для создания хешей блоков. Даже малейшее изменение данных полностью меняет хеш.

Децентрализация
В отличие от централизованных баз данных, блокчейн распределён между множеством узлов (нодов). Каждый узел хранит полную копию блокчейна. Это обеспечивает:
- Отказоустойчивость - нет единой точки отказа
- Прозрачность - все транзакции видны
- Безопасность - изменить данные крайне сложно

Консенсус
Механизмы консенсуса позволяют узлам договориться о состоянии сети:
- Proof of Work (PoW) - майнинг, требует вычислительной мощности (Bitcoin)
- Proof of Stake (PoS) - валидация на основе доли владения (Ethereum 2.0)
- Delegated Proof of Stake (DPoS) - делегированная валидация
- Proof of Authority (PoA) - доверенные валидаторы

Майнинг
Процесс создания новых блоков в блокчейне. Майнеры решают сложные математические задачи для добавления блока. За это они получают награду в криптовалюте.

Сложность майнинга
Автоматически регулируется для поддержания постоянного времени создания блока (10 минут для Bitcoin). Если больше майнеров - сложность растёт.

Криптовалюты
Цифровые валюты, работающие на блокчейне:

Bitcoin (BTC)
Первая и самая известная криптовалюта, создана в 2008 году Сатоши Накамото. Максимальное количество: 21 миллион монет. Используется Proof of Work.

Ethereum (ETH)
Платформа для смарт-контрактов и децентрализованных приложений (dApps). Поддерживает создание токенов стандарта ERC-20. Перешёл на Proof of Stake.

Смарт-контракты
Программы, автоматически выполняющиеся при соблюдении условий. Написаны на Solidity (для Ethereum). Используются в DeFi, NFT, DAO.

DeFi (Децентрализованные финансы)
Финансовые сервисы без посредников:
- Lending/Borrowing - кредитование
- DEX - децентрализованные биржи (Uniswap, PancakeSwap)
- Staking - блокировка токенов для получения дохода
- Yield Farming - предоставление ликвидности

NFT (Non-Fungible Token)
Уникальные токены, представляющие право собственности на цифровой актив. Используются для:
- Цифрового искусства
- Игровых предметов
- Виртуальной недвижимости
- Коллекционных карточек

Проблемы блокчейна
- Масштабируемость - ограниченная пропускная способность (Bitcoin: 7 tps, Ethereum: 15 tps)
- Энергопотребление - PoW требует огромных энергозатрат
- Регуляция - неопределённость правового статуса
- 51% атака - контроль большинства узлов

Решения для масштабирования
- Lightning Network - второй уровень для Bitcoin
- Sharding - разделение блокчейна на части
- Rollups - объединение транзакций
- Sidechains - параллельные цепочки

Web3
Концепция децентрализованного интернета на основе блокчейна. Пользователи владеют своими данными. Работает через кошельки (MetaMask, WalletConnect).

DAO (Decentralized Autonomous Organization)
Организации, управляемые через смарт-контракты. Решения принимаются голосованием держателей токенов. Нет централизованного управления.

Популярные блокчейны
- Bitcoin - цифровое золото
- Ethereum - смарт-контракты и dApps
- Binance Smart Chain - быстрые и дешёвые транзакции
- Solana - высокая скорость (65000 tps)
- Cardano - научный подход, PoS
- Polkadot - кросс-чейн взаимодействие

Будущее блокчейна
Применение в логистике, здравоохранении, управлении цепочками поставок, цифровой идентификации, голосовании.
